Code Cracker

Can you complete the code?

Instructions

The computer will generate a sequence of numbers from 1 to 5. It can only use each number once per round.

For level 1 there will be one break in the code for you to work out the missing number. For level 2 there will be 2 missing numbers and so on.

If you can complete the game you can put yourself on the leaderboard.
